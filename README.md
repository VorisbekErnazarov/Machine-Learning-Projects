# Machine-Learning-Projects

This repository contains several machine learning and deep learning projects based on satellite images and remote sensing data.

## Projects

- Road Detection
- Cloud Detection
- Cloud Removal and Behind-Cloud Analysis
- Land vs Water Classification
- Forest vs Non-Forest Classification

## Datasets

Some projects use Sentinel-1, Sentinel-2 and SEN12MS datasets.

## Tools and Libraries

- Python
- Jupyter Notebook
- NumPy
- Pandas
- OpenCV
- Matplotlib
- PyTorch
- Scikit-learn
- Segmentation Models PyTorch
- Rasterio
- Albumentations

## Repository Structure

```
Machine-Learning-Projects
│
├── road_detection.ipynb
├── cloud_detection.ipynb
├── cloud_detection_&_behind_cloud_analysis_project.ipynb
├── land_water.ipynb
└── forest_non_forest.ipynb
```

Each notebook contains data preparation, model training and evaluation for a specific task.
