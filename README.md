# Machine-Learning-Projects

This repository contains machine learning and deep learning projects focused on satellite imagery analysis and remote sensing applications.

## Projects Included

- Road Detection from Satellite Images
- Cloud Detection
- Land vs Water Classification
- Forest vs Non-Forest Classification

## Technologies Used

- Python
- Jupyter Notebook
- NumPy
- Pandas
- OpenCV
- PyTorch
- Scikit-learn
- Segmentation Models PyTorch

## Purpose

These projects were developed for practice and research in computer vision, remote sensing, and geospatial AI tasks.
